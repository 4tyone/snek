# Git Status

Show current git repository status.

## Steps

1. Get git status:

```bash
.snek/scripts/get-git-status.sh
```

2. Display in readable format:
   - Current branch
   - Ahead/behind remote
   - Staged files
   - Modified files
   - Untracked files
