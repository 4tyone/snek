# Session Info

Show detailed information about the currently active snek session.

## Steps

1. Get active session info:

```bash
.snek/scripts/get-active-session.sh
```

2. Display:
   - Session ID
   - Session name
   - Max tokens setting
   - List of context files
   - Number of code snippets
   - Last updated time
